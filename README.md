## Getting Started

To get started with To-Do Pro, follow these instructions to set up the project locally.

### Prerequisites

- Node.js
- npm
- AWS account
- Amplify CLI

1. Clone the repo

```sh
git clone
```

2. Install NPM packages

```sh
npm install
```

3. Set up Amplify

```sh
amplify init
```

4. Add the API

```sh
amplify add api
```

5. Push the changes

```sh
amplify push
```

6. Start the development server

```sh
npm run dev
```
